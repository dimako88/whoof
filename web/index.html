Apple Health Sync

whoof runs entirely on your phone — these import your Apple Health data with no server and nothing leaving the device.

📄 Import export.xml (full history)

  1. iPhone Health app → profile photo → Export All Health Data.
  2. Unzip the file, then tap 📄 Import and pick export.xml.

Reads Body Mass, Height, Resting HR, VO₂ Max, Steps and more. Parsed on-device.

📱 One-tap Shortcut (live values)

Install a Shortcut that reads the metrics you want and opens …/#health=<data> back into whoof. Then tap 📱 iPhone any time to pull the latest. Works in Safari and Bluefy.

Advanced: live LAN sync via Health Auto Export
  1. Run ./run.sh dash --host 0.0.0.0 on this Mac.
  2. In Health Auto Export → Automations → REST API, paste:
  3. Pick Body Mass, Height, Resting Heart Rate, VO2 Max and save. Only works while the Mac server is running on the same Wi-Fi.
